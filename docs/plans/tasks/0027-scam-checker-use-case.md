# Task 0027: Scam Checker Use Case

## Problem
Users receive suspicious messages, emails, calls, and offers but aren't sure if they're scams. There's no easy, non-technical way to get a quick assessment without Googling or calling someone.

## Solution
Add a new "Is This a Scam?" use case that walks users through describing the suspicious contact and gives a clear verdict with next steps.
