# One-Off Tasks

Standalone tasks that don't belong to a phase. Each task gets a file here.

## Naming Convention

`NNNN-short-description.md` — sequential numbering.

## Template

```markdown
# Task NNNN: Short Title

**Status:** TODO | IN_PROGRESS | DONE
**Requested:** YYYY-MM-DD
**Completed:** YYYY-MM-DD
**PR:** #NN

## Description
What needs to happen.

## Acceptance Criteria
- [ ] Criteria 1
- [ ] Criteria 2

## Notes
Implementation notes, decisions, context.
```
